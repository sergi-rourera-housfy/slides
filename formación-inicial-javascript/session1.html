🔄 Variables: let vs const

// const = no cambia (usa esto por defecto)
const IVA = 0.21;
const nombreEmpresa = "MiEmpresa";

// let = puede cambiar
let contador = 0;
contador = contador + 1; // Ahora es 1

// ❌ NUNCA uses var (está obsoleto)
Regla de oro: Usa const siempre, let solo si necesitas cambiar el valor

⚖️ Comparaciones (== vs ===)

// === compara valor Y tipo (USA ESTE)
5 === 5        // true
5 === "5"      // false (número vs string)

// == convierte tipos (evítalo)
5 == "5"       // true (confuso)

// Otros operadores
10 > 5         // true
10 <= 10       // true
"hola" !== "adiós"  // true (son diferentes)

🏋️ EJERCICIO PRÁCTICO 1

Contexto: Tienes una lista de productos con precios sin IVA. 

// INPUT
const productos = [
  { nombre: "Teclado", precio: 50 },
  { nombre: "Ratón", precio: 25 },
  { nombre: "Monitor", precio: 200 }
];

// TU MISIÓN:
// 1. Añadir IVA (21%) a cada precio
// 2. Devolver nuevo array con los precios actualizados

✅ SOLUCIÓN EJERCICIO 1

const productos = [
  { nombre: "Teclado", precio: 50 },
  { nombre: "Ratón", precio: 25 },
  { nombre: "Monitor", precio: 200 }
];

// Solución simple (veremos mejores formas en sesión 3)
const productosConIVA = [
  { 
    nombre: productos[0].nombre, 
    precio: productos[0].precio * 1.21 
  },
  { 
    nombre: productos[1].nombre, 
    precio: productos[1].precio * 1.21 
  },
  { 
    nombre: productos[2].nombre, 
    precio: productos[2].precio * 1.21 
  }
];

return productosConIVA;
